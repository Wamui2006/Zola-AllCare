# Zola AllCare Website 
Zola AllCare is a psychosocial and emotional wellness platform that provides support systems through Z-Talk, Z-Space, and Z-Community. 
 
## ?? Overview 
Zola AllCare creates safe spaces and sustainable mental wellness programs for individuals, institutions, and organizations. The website showcases Zola's mission, service plans, and contact channels. 
 
## ?? Project Structure 
- index.html: Home page introducing Zola AllCare. 
- about.html: Background and mission details. 
- z-talk.html: Describes Z-Talk therapy and mentorship services. 
- z-space.html: Details on physical, smart, and connect spaces with embedded PDFs. 
- z-community.html: Explains community programs for all age groups. 
- contact.html: Contact form and booking details. 
- assets/: Contains CSS, JS, images, and PDFs. 
 
## ?? Technologies Used 
- HTML5 and CSS3 for structure and styling 
- JavaScript for interactivity (cookie banner, mobile nav) 
- PHP for backend contact and configuration files 
- MySQL for database connectivity (via config.php) 
 
## ?? Deployment 
Hosted locally via Laragon or XAMPP. To view: place folder in `www` or `htdocs` and visit http://localhost/Zola.
